# カード実装リスト（身内語録大戦）

**凡例**


| 記号  | 意味             |
| --- | -------------- |
| ✓   | 完了・該当          |
| —   | 未着手・該当なし       |
| 未検証 | 実装済みだがプレイテスト未了 |


**デッキルール**: 40枚 / 同一カード最大4枚 / 初期手札5枚 / ターン開始ドロー1枚（山札切れ敗北）

## 運用メモ

- **実装済み**: JSON ファイルがあり `manifest.json` に登録されている
- **バグ無し**: 対戦・コンボ・条件効果を手動確認後に ✓ に更新
- **実装除外**: ゲームに入れないカードは ✓（他列は —）
- **コンボ効果**に「コスト-1（標準）」とある場合、`speaker_effect` 未指定時のデフォルト動作を指す
- 新カード追加時は `CARD_FORMAT.md` のスキーマに従い、本表に行を追加する
